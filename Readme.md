# Museo

  Museo font component using @font-face. The font includes 300, 500 and 700 weights.
  
  http://www.fontsquirrel.com/fonts/Museo?q=museo

## Installation

  Install with [component(1)](http://component.io):

    $ component install esundahl/museo

## API

```css
body {
  font-family: 'Museo';
}
```
